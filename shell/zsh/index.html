zsh¶

Versão 1 - atualizada em 05/05/2017

Comando para instalar o zsh nos sistemas operacionais Arch Linux e Manjaro Linux:

$ sudo pacman -S zsh

Comando para instalar o zsh nos sistemas operacionais Debian, Ubuntu, Linux Mint:

$ sudo apt install zsh

Comandos para tornar o zsh o shell padrão para o seu usuário:

$ sudo usermod -s /bin/zsh seu_usuario

ou

$ chsh -s /bin/zsh

oh-my-zsh¶

O oh-my-zsh é um framework que facilita a configuração do shell ZSH.

Comandos para instalar o oh-my-zsh em qualquer sistema operacional Linux:

  • Framework
$ sh -c "$(wget https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh -O -)"
  • Fontes com icones para o terminal (opicional)
$ git clone https://github.com/powerline/fonts.git
$ cd fonts
$ ./install.sh
$ cd ..
$ rm -rf fonts
